# fpy1101-evaluacion-4
# Evaluación 4 - Programación

Este proyecto corresponde a la **Evaluación 4 de Programación**. El programa permite administrar una biblioteca mediante un menú interactivo, ofreciendo las siguientes opciones:

* Agregar un libro.
* Buscar un libro.
* Eliminar un libro.
* Actualizar la disponibilidad de un libro.
* Mostrar todos los libros registrados.
* Salir del sistema.

## Validaciones implementadas

El programa realiza las siguientes validaciones para asegurar el correcto ingreso de los datos:

* El título del libro no puede estar vacío.
* La cantidad de copias debe ser un número entero mayor o igual a **0**.
* El período de préstamo debe ser un número entero mayor que **0**.
